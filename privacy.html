Summarize This Privacy Notice

Summarize This is a static Trello Power-Up. It reads Trello card context only when a user opens the Power-Up popup or runs a setup flow.

Data handled

  • Card title, description, labels, members, list, board, due date, checklists, comments, activity, attachment metadata, custom fields, and private user feedback may be used to produce card intelligence.
  • Analysis history, review state, feedback, budget records, and export records are stored in Trello member-private Power-Up storage where available.
  • Provider API keys are intended for Trello member-private storage only. Local preview mode does not persist provider API keys.

AI providers

If AI mode is enabled, selected card context may be sent to the configured provider or to the configured backend proxy. Sensitive-card handoff can require explicit approval before provider calls.

Trello writes

The Power-Up does not silently post summaries or edit cards. Trello comment drafts require review and explicit approval before posting when the Trello runtime supports posting.

Contact

For privacy questions, use the support contact configured in the Trello Power-Up manifest.
