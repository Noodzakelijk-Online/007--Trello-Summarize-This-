# Technical Debt Register

Date: 2026-07-12

## High Priority

- Binary attachment extraction remains metadata-only for PDF, Word, Excel, and image files.
- Disconnected backend/admin files remain in the repo without a verified shipped role.
- No automated browser-level end-to-end suite exists for the Trello runtime.
- Historical milestone docs still exist and can confuse readers if they skip the audit docs.

## Medium Priority

- No one-command local development flow.
- No formal feature flag system.
- No measured accuracy benchmark.
- No production-grade release/canary/rollback discipline for a unified deployable system.

## Low Priority

- Legacy modules and summaries could be reorganized into a clearer archive area.
- Additional UX help/onboarding could reduce operator confusion.
