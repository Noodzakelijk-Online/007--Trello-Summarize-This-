# Autonomy-First Product Review

Date: 2026-07-23 (Phase 082)

## Evaluation Criteria

1. **Human Operator Control:** Automated actions (such as Trello comment posting) require explicit human review and approval.
2. **Local Autonomy:** Local rule-based summarization operates independently of external API availability.
3. **Deterministic Fallbacks:** When AI providers fail or timeout, the system automatically falls back to local rules without crashing.
