# Responsive and Browser Compatibility

Date: 2026-07-23 (Phase 050)

## Supported Environments

- **Browsers:** Chrome 100+, Firefox 100+, Safari 15+, Edge 100+.
- **Trello Clients:** Trello Web Client (Desktop & Mobile Browser), Trello Desktop App (macOS & Windows via Electron wrapper).
- **Responsive Layout:** Popup UI adapts dynamically to Trello iframe modal width (`t.sizeTo('#app')`).
