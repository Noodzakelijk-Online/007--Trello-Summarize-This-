# User Onboarding and First-Run Wizard

Date: 2026-07-23 (Phase 105)

## First-Run Experience

When a user launches the Power-Up for the first time without prior configuration:

1. **Welcome Notice:** A modal banner introduces the 3-step workflow (Analyze → Review → Export).
2. **Local Summary Ready:** Explains that built-in local summarization works immediately without API keys.
3. **AI Setup Option:** Provides direct link to Settings panel for entering OpenAI, Anthropic, or Google AI keys.
4. **Member Privacy Guarantee:** Explains that data stays in member-private storage and card content is never stored on external servers.
