# Domain Model Specification

Date: 2026-07-23 (Phase 060)

## Domain Entities

1. **Card Context:** The immutable snapshot of a Trello card (title, description, checklists, comments, attachments, custom fields) read during analysis.
2. **Analysis Result:** The normalized output generated by either an AI provider or local rules (about, blockers, decisions, next steps, evidence, risks, quality score).
3. **Ledger Run:** The private audit record of an analysis run stored in member-private storage.
4. **Export Packet:** A formatted representation of an analysis run (Markdown, JSON, VA handoff, change brief, comment draft).
5. **Settings Configuration:** User preferences for AI providers, proxy URL, output language, token budget, and templates.
