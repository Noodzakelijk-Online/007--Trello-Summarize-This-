# Post-Completion Maintenance Plan

Date: 2026-07-12

## Ongoing Tasks

- Re-run automated tests after every popup, summarizer, or attachment-processing change
- Re-run live Trello manual verification after changes to comment posting, settings storage, or provider handoff logic
- Keep audit docs aligned with the shipped scope

## Triggered Reviews

- If binary attachment extraction is added, update the security, privacy, and acceptance docs
- If backend/admin files are reactivated, add dependency, runtime, security, and test evidence before expanding completion claims
- If new provider modes are added, verify failure and fallback behavior explicitly
