# Operator Safety Stop And Emergency Controls

Date: 2026-07-12

## Current Safety Stops

- Sensitive-card approval gate before broader AI handoff
- Approval gate before Trello comment posting
- Local fallback when provider/proxy paths fail
- Metadata-only default for binary attachments

## Recommended Future Safety Stops

- Explicit global read-only mode if future shared writeback features are added
- Operator-visible maintenance mode if a backend/admin platform is ever reactivated
