Configure Trello Power-Up

Hosted URL needed for Trello Trello loads Power-Ups from its website, so the connector URL must be an HTTPS address that Trello can reach. The Windows app can run the standalone tool locally, but Trello cannot use a private local address as the Power-Up connector.
